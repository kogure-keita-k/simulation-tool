## 画面1：試算入力（サイドバー左側）
検針票をもとに試算するための入力画面です。※印は最低限必要な項目。

### メニュー / ナビゲーション
1. メニューボタン群 … 「見積書」「削減レポ」「結果一覧」「各種設定（管理画面）」など各機能へ移動します。

### 顧客情報
2. ※顧客名（御中） … 提案先の会社名。提案書・見積書・削減レポの宛名に使われます。
3. ※業態 … 製造/倉庫/事務所などから選択。業態ごとの「空調のピーク比率・使用量比率」を試算に使います。該当が無い場合は「その他（数値を指定）」で比率を手入力できます。

### 検針票データ
4. ※検針票の取込（Upload） … 電力会社の検針票データ（CSV / Excel）を取り込みます。12ヶ月分の「使用量・最大需要電力（デマンド）・契約電力・各単価」を使用します。
5. サンプル値代入 … データが無いとき、サンプルの12ヶ月値を自動入力して動作確認できます。
6. インポート用テンプレDL … 取込用のExcelテンプレートをダウンロードできます（列名がこの形式なら確実に取り込めます）。
7. 電力会社プリセット … 単価が不明なとき、電力会社の目安単価を全月へ一括反映できます。

### 室外機情報
8. ※室外機 総台数 / 制御対象台数 … 施設の室外機台数と、そのうち制御対象にする台数を入力します（制御対象率が自動計算されます）。
9. 機材情報なし → 検針票から概算する（トグル） … 室外機・機材情報が無い場合にON。検針票から台数を自動推定します（詳細は「概算モード」章）。
10. 機体（型番）リストの取込 … 型番付きの室外機リストを取り込むと、定格×稼働で制御可能比率を自動精緻化します。

### 初期費用
11. 詳細見積（見積書方式）トグル … 電材費・作業人数・日数などの詳細がある場合に、見積書と同じ方式で算出します（OFFは従来の概算）。
12. 材料費 / 工事費 … 初期費用の内訳を入力（概算モードでは標準単価×台数で自動表示）。
13. 交通費（概算） … 台数×単価×エリア係数で自動計算されます。
14. ハードウェア関連費 … 制御機器の費用（部材リストから自動、または概算モードでは標準単価で自動）。
15. 空調システム構築費 … 単価×台数で自動加算されます。

### 算出オプション
16. 季節性（ベースロード法） … 最少月を基礎電力とみなして空調分を抽出する方式に切替。
17. 業態を考慮 … 基礎電力の一部を空調分として残す配慮（過小評価の回避）。

### 実行
18. ※試算実行ボタン … 入力後にこのボタンで計算し、結果ダッシュボードを表示します。

## 画面2：結果ダッシュボード（試算実行の直後に表示）
試算結果の確認と、提案書・試算表などの出力を行う画面です。以下は上から順に出る項目です。

### 冒頭のチェック・注釈
1. 入力整合性チェック … 入力の妥当性を自動点検。「1台あたり初期費用が高すぎる」等の警告や、異常なしの表示が出ます。
2. 算出の前提（注釈） … 「概算・業態標準値・自動補完」など、この結果が依拠する仮置き前提の一覧。提案書PPTXにも同じ注釈が入ります。
3. 概算：3シナリオ比較（概算モード時のみ） … 「ベスト／標準／保守」の3案を1表で比較（予測台数・制御比率・年間削減・初期費用・投資回収）。想定の幅を確認できます。

### KPI（重要指標）×4
4. 初期費用（総投資） … 機材・工事込みの初期投資額。
5. 投資回収期間 … 年間純利益ベースで初期費用を回収するまでの年数。
6. 年間削減額 … 基本料金削減＋電力量（従量）削減の年間合計。
7. N年累計削減（NPV併記） … 期間（既定10年）の累計削減額と、割引後の正味現在価値（NPV）。

### 要約・グラフ・比較
8. AIサマリー … 結論を先出しで要約したコメント（結論ファースト）。
9. 主要グラフ … 月別の電力使用量＋最大デマンドなどの推移グラフ。
10. 比較表 … 現状維持 vs 導入後 の月別・項目別の比較表。

### 詳細分析（タブ切替）
11. 「グラフ」タブ … 最大デマンド／電力使用量／電気料金を切替表示。月別データも展開できます。
12. 「財務」タブ … NPV・IRR など投資判断の財務指標。
13. 「制御強度」タブ … 制御モード（どの程度制御するか）の説明・設定。
14. 「リスク」タブ … 10年スパンの財務リスクの確認。
15. 「算出方法」タブ … 削減額の算出根拠（計算式と内訳）を提示。
16. 「回収逆算」タブ … 希望する投資回収年数から、必要な制御強度・削減量を逆算。

### 付加価値・出力
17. 付加価値・財務換算 / 1枚サマリー … 削減額を本業（主力製品）の売上・利益に換算。1枚要約（ペライチ）も確認できます。
18. 提案書・出力：PowerPointダウンロード … マッキンゼー風の提案書（PPTX）を出力。概算モードなら概算注記・概算説明ページ付き。
19. 提案書・出力：Excelダウンロード … 前提・計算・顧客提示用サマリーの試算表（XLSX）。
20. 結果一覧に保存 … この案件を保存（企業データにも月次を累積）。後日、削減レポ等で再利用できます。
21. 営業トークスクリプト … 商談で使えるトーク例を表示。
22. スライド貼付用 画像出力（PNG） … グラフ・KPI・制御可否リスト等を、資料貼付用のPNGで個別に出力。

## 画面3：各種設定（管理画面）
計算に使う単価・係数・マスタを管理します（通常は初期値のままで問題ありません）。タブ構成です。

1. 計算ロジック変更 … 容量削減率、各種固定値（換算係数・CO2係数・力率基準など）、および「概算マスタ」（標準室外機定格kW・標準機材費/台・標準工事費/台・HW費・概算制御比率）を編集。
2. 機材単価マスタ（見積） … 見積書の機材費に使う商品ごとの単価。
3. 馬力・機材テーブル … 型番→定格kW などの対応表、室外機1台あたりの機材セット。
4. 業態マスタ設定 … 業態ごとの空調割合（最大デマンド比率・使用量比率）。
5. 表示項目変更 … 画面・出力に表示する項目の切替。
6. 変更・DL履歴ログ … 設定変更やダウンロードの履歴。
7. 画像出力（手動値） … 任意の数値を指定してKPI画像等を生成。

## 画面4：見積書
機材・工事の見積書を作成し、PDF / Excel で出力します。

1. 件名 … 見積の件名。
2. 宛先（御中） … 顧客名（試算画面から引用）。
3. 担当者 / 書類番号 / 納期 / 納品先 / 支払条件 … 見積の諸条件。
4. 明細（機材・工事・ハードウェア） … 品目・数量・単価・金額（単価はマスタから引用）。
5. 印①（左）／印②（右） … 押印欄の氏名を左右それぞれ入力（空欄なら枠のみ）。
6. 出力（PDF / Excel） … 見積書をダウンロード。

## 画面5：削減レポ（電力削減実績報告書）
運用後の実績を、客先様式の削減実績報告書として作成します（基準年 vs 対象年の比較）。

### データの取り込み
1. 企業の蓄積データから呼び出す … 過去のシミュ保存・インポートで自動蓄積された月次を、企業を選ぶだけで呼び出し。
2. 結果一覧の保存案件から呼び出す … 保存済み案件の月次データを呼び出し。
3. 明細Excelを追加インポート … 全体把握等のExcelを取り込み（前年/対象年の不足分を補う）。
4. インポート用テンプレDL … 取込用テンプレート。列名が異なると取り込めないため、これを使うと確実。

### 対象期間・ラベル
5. 顧客名（御中） … 報告書の宛名。
6. 対象期間の開始月（3ヶ月） … 対象とする3ヶ月の起点。基準年は「前年同月」を自動採用（変更は折りたたみ内）。
7. 発行日 … 報告書の発行日。
8. 表示ラベル（客先様式） … 「基準年同月」「削減効果」等のラベルを客先の表記に合わせて調整・保存。

### 特殊モード
9. エクセル眼鏡はこちら（営業日数補正モード・トグル） … 眼鏡様専用。使用電力量の削減効果を「対象年を基準年の営業日条件に換算」して算出。基準年・対象年の各月営業日数を入力します。通常の会社はOFFのまま（単純差）。

### プレビュー・出力
10. 数値プレビュー … ①利用状況／②最大需要 比較／③使用電力量 比較を、基準年 vs 対象年の2年比較で表示。表は手修正も可能。※2年比較には両年のデータが必要です。
11. グラフプレビュー … ②③のグラフ（基準年 vs 対象年）。
12. 出力（PDF / Excel） … 削減実績報告書をダウンロード。

## 画面6：請求書（成果報酬型）
削減実績を引用し、成果報酬型の請求書を作成します。

1. 成果報酬率 … 例：0.5＝50%。削減額のうち報酬とする割合。
2. 契約電力 削減実績報酬 … 契約電力の削減量（削減レポから引用）× 基本料金単価 × 報酬率。
3. 使用電力量 削減実績報酬 … 使用電力量の削減量 ×（電力量単価＋燃料調整費＋再エネ賦課金）× 報酬率。
4. 各種諸条件 … 発行日・伝票番号・支払期日・件名・登録番号（インボイス）など。
5. 押印（※印）の氏名 … 左右それぞれ入力。
6. 振込先 … 銀行・支店・口座・名義。
7. 出力（PDF / Excel） … 請求書をダウンロード。

## ＜横断＞概算モードの使い方（機材情報が無いとき）
室外機・機材情報が無くても、検針票と業態標準値から概算できます。※算出式（削減量・電気料金・回収）は通常試算と同一で、台数と初期費用の「入力」を推定するだけです。

1. 入口 … 画面1「室外機情報」の「機材情報なし→検針票から概算する」トグルをON。
2. 台数の自動推定 … 最大デマンド × 業態の空調ピーク比率 ÷ 標準室外機定格kW で室外機台数を推定（画面上で調整可・再推定ボタンあり）。
3. 初期費用も概算 … 材料費・工事費・ハードウェア費を「標準単価 × 推定台数 ＋ 固定費」で自動算出。
4. 3シナリオ比較 … 結果画面でベスト／標準／保守の幅を確認。
5. 資料への注記 … 提案書PPTXには概算注記と「概算の算出ロジック」説明ページが自動で入ります。
6. 単価・係数の調整 … 画面3「各種設定→計算ロジック変更」の概算マスタで、標準室外機kW・各単価・制御比率を調整できます（自社実績値に更新すると精度が上がります）。
