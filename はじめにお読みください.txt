■ DPS 空調デマンド制御 試算アプリ — 起動のしかた

【いちばん簡単な起動】
　このフォルダの「起動.bat」を ダブルクリック するだけです。
　・初回だけ必要なライブラリを自動でインストールします（数分／ネット接続が必要）。
　・準備ができるとブラウザが自動で開き、アプリが表示されます。
　・開かない場合はブラウザで  http://localhost:8501  を開いてください。
　・終了するときは、黒い画面（コマンド画面）で Ctrl+C を押すか、画面を閉じます。

【事前に必要なもの】
　・Python 3.9 以上（未インストールの場合のみ）
　　https://www.python.org/downloads/ からインストール。
　　※インストール時「Add Python to PATH」に必ずチェックを入れてください。
　・インターネット接続（初回セットアップ時のみ）

【配布・コピーするとき】
　・このフォルダ「まるごと」を渡してください（中のファイルは分けないでください）。
　　app.py / slide_image_export.py / scripts フォルダ / 各種アイコン画像 /
　　requirements.txt / 起動.bat などが揃っている必要があります。
　・設定や保存データは dps_app_data.json に入ります。
　　まっさらな状態で使わせたい場合は、このファイルを削除してから渡してください。

【デスクトップから起動したい場合】
　「起動.bat」を右クリック →「ショートカットの作成」→ できたショートカットを
　デスクトップに移動しておくと、次回からデスクトップのアイコンで起動できます。

【うまくいかないとき】
　・「Python が見つかりません」→ 上記の Python をインストール（PATH追加にチェック）。
　・ライブラリのインストールで失敗 → ネット接続を確認して、もう一度 起動.bat を実行。
　・それでも直らない場合は、黒い画面のエラー文言を控えて管理者へご連絡ください。
