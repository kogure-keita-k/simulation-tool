  • 1Upload（青ボタン）：検針票Excel/CSVを選択。列を自動認識して表に反映（「7/7認識」表示）

導入手順・FAQ

初回セットアップ（Windows）

  1. Python 3.9以上を python.org から導入（「Add python to PATH」にチェック）。※コマンドは py -3。
  2. アプリのフォルダで初回のみ：
    # 必要ライブラリ（初回だけ）
py -3 -m pip install -r requirements.txt
  3. 起動：start.bat をダブルクリック、または 
    py -3 -m streamlit run app.py
    ブラウザが自動で開きます（開かなければ http://localhost:8501）。停止は Ctrl + C。
起動できない／不安定：「Starting the CLR failed…」等が出ても一時的です。ターミナルを開き直すか PC再起動で解消します。

よくある質問

  • 台数・機材費が分からない → STEP3の概算モードON。標準単価で初期費用まで自動概算。
  • 「列名が違う」と出る → STEP1のテンプレDLを使うか、検針年月を 2025/04/202504 形式に。
  • 入力をやり直したくない → メニュー直下のスナップショットで保存→呼び出し。
  • 設定を変えたのに反映されない → アプリ（Streamlit）を再起動すると確実です。
